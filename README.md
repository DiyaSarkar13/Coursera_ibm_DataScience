This project was completed as part of the IBM Data Science Professional Certificate on Coursera. The objective is to predict whether a SpaceX Falcon 9 first-stage booster will successfully land, which helps estimate launch costs and reusability potential.

Project Components:

Data collection using the SpaceX REST API
Web scraping launch data from Wikipedia
Data cleaning and preprocessing with Pandas
Exploratory Data Analysis (EDA) using Matplotlib, Seaborn, and Plotly
SQL analysis using SQLite/IBM Db2
Interactive geospatial visualizations with Folium
Interactive dashboard development using Plotly Dash
Machine learning models including Logistic Regression, SVM, Decision Tree, and KNN
Model evaluation using accuracy, confusion matrix, and cross-validation

Results:

Analyzed 90 Falcon 9 launches
Achieved 83.3% prediction accuracy
Identified key factors affecting landing success, including launch site, payload mass, orbit type, and booster version.
Resume Bullet
